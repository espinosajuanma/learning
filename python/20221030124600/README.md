# Types

## Numbers

- Integer - `int`
- Float - `float`
- Complex - `complex`

## Booleans

- Boolean - `bool`

## Sequences

- String - `str`
- List - `list` - `[1, 2, 3]`
- Tuple - `tuple` - `(1, 2, 3)`

## Sets

- Set - `set` - `{1, 2, 3}`

## Dictionaries

- Dictionary - `dict` - `{1: '1', 2: '2'}`
